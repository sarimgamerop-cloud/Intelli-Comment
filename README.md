## 🤖 What is IntelliComment?

IntelliComment is an open-source project based on **Tesseract Optical Character Recognition (OCR)** technology rather than using old-fashioned HTML parsing.

⚡ This program provides a simple command-line interface and is easy to set up on almost any modern computer.

---

## 🧠 Comment Generation

Instead of generating irrelevant or machine-heavy comments, IntelliComment uses **free API keys** for smarter processing.

💡 If you already own a paid API key, you get enhanced performance and usage limits.

Free API keys also work for normal usage, and using multiple keys can help improve overall limits and flexibility.

---

## Installation

The program can can work on both **Windows** ( Windows 10/11 Recommended ) and **Linux** ( Make sure required dependencies can be installed.)

**Start of by Cloning the Project :**
```
git clone https://github.com/sarimgamerop-cloud/Intelli-Comment.git
cd Intelli-Comment
```
**Install the Dependencies :**
```
 pip install -r requirements.txt
```
